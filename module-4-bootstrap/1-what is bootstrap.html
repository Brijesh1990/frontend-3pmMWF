what is bootstrap ? bootstrap : bootstrap is a framework of website designing and development. It is built on top of HTML, CSS, and JavaScript. Bootstrap is a responsive and mobile-first framework, meaning it adjusts its layout and styling to fit different screen sizes and devices. It provides pre-built components, utilities, and grid system that can help you build websites quickly and easily. Bootstrap is widely used by developers and designers to create responsive and user-friendly websites. key features of bootstrap : bootstrap provides various features and components to help you build websites more efficiently and effectively. Some of the key features include: - Responsive design: Bootstrap uses CSS media queries to create responsive layouts that adapt to different screen sizes and devices. - Flexbox grid system: Bootstrap provides a flexible and responsive grid system that allows you to create responsive and flexible layouts by using columns and rows. - Pre-built components and utilities: Bootstrap provides pre-built components, utilities, and grid system that can help you build websites quickly and easily. - Components: Bootstrap provides various components, such as buttons, forms, navigation, and tables, that can be used to create visually appealing and functional websites. - JavaScript plugins: Bootstrap provides various JavaScript plugins, such as tooltips, popovers, and modals, that can help you add additional functionality to your websites. - Documentation and examples: Bootstrap provides documentation and examples that can help you learn how to use its components and features. - Community and support: Bootstrap has a strong community and support, with numerous online forums, resources, and tutorials available to help you learn and troubleshoot issues. examples : https://getbootstrap.com/ resources : https://www.w3schools.com/bootstrap/ tutorials : https://www.youtube.com/watch?v=Z9qN71j-Q34&list=PL6B39D2168B7A63D1 community : https://github.com/twbs/bootstrap/issues support : https://stackoverflow.com/questions/tagged/bootstrap documentation : https://getbootstrap.com/docs/4.5/getting-started/introduction/ examples : https://getbootstrap.com/docs/4.5/examples/ community : https://github.com/twbs/bootstrap/issues support : https://stackoverflow.com/questions/tagged/bootstrap documentation : https://getbootstrap.com/docs/4.5/getting-started/introduction/ how to use bootstrap ? a) online CDN b) offline CDN c) using bootstrap npm package d) using bootstrap yarn package e) using bootstrap bower package f) using bootstrap composer package
